# 🧾 Pro Invoice Generator

> A modern, fully responsive invoice generator built with React. Create, customize, and download professional PDF invoices instantly.

## 📝 Description

Pro Invoice Generator is a lightweight, client-side React application designed to make billing simple. Recently modernized from legacy class components to modern React Hooks, this tool provides a sleek UI for businesses or freelancers to generate itemized invoices. It calculates subtotals, taxes, and discounts in real-time and allows users to export high-resolution PDFs directly from the browser without needing a backend.

## ✨ Features

* **Modern React Architecture:** Built entirely with Functional Components and Hooks (`useState`) for fast, reliable state management.
* **Real-Time Calculations:** Automatically calculates item totals, subtotals, configurable tax rates, and discount percentages.
* **High-Resolution PDF Export:** Generates crisp, professional PDFs using `jsPDF` and `html2canvas`.
* **Multi-Currency Support:** Easily toggle between USD, GBP, JPY, CAD, AUD, SGD, CNY, and BTC.
* **Responsive Design:** A polished, mobile-friendly interface powered by React Bootstrap and custom CSS.

## 🛠️ Technologies Used

* **Frontend:** React.js (Hooks)
* **Styling:** React-Bootstrap, Bootstrap 5, Custom CSS
* **Icons:** React Icons (`react-icons/bi`)
* **PDF Generation:** `jsPDF`, `html2canvas`
